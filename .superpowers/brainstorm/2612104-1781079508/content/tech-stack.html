Технологический стек

Для Windows антивируса нужна хорошая интеграция с ОС. Какой стек предпочитаешь?

A

C# + .NET (WPF / MAUI)

Нативный Windows-стек. Отличный доступ к WinAPI, реестру, службам. WPF — мощный UI, много готовых библиотек для безопасности. Рекомендую как основной выбор.

Плюсы

  • Родной для Windows
  • Богатый WinAPI доступ
  • NuGet: YARA, ClamAV обёртки, Authenticode
  • Хорошая производительность

Минусы

  • Только Windows (WPF)
B

Python + PyQt / PySide6

Быстрая разработка, огромная экосистема. YARA-python, python-magic, pywin32 — всё есть. Хуже для системного уровня, медленнее, сложнее дистрибьютировать.

Плюсы

  • Быстрая разработка
  • Огромная экосистема
  • YARA-python, pywin32

Минусы

  • Медленнее нативных
  • exe-дистрибуция громоздкая
  • Ограниченный низкоуровневый доступ
C

Electron / Tauri (JS/TS + Rust)

Современный красивый UI на веб-технологиях. Tauri — лёгкий Rust бэкенд + React фронтенд. Electron — тяжелее, но проще. Системная интеграция через нативные модули.

Плюсы

  • Красивый современный UI
  • Kросс-платформа
  • Большое сообщество

Минусы

  • Electron тяжёлый (150+ MB)
  • Системный уровень через отдельный процесс
