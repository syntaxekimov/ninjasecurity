Целевая платформа

Это самый важный выбор — от него зависит весь стек технологий и возможности приложения

A

Windows

Самая актуальная платформа для антивируса. Богатый API (WinAPI, WFP, minifilter), доступ к реестру, службам. 90%+ всех вирусов нацелены на Windows.

Плюсы

  • Максимальная востребованность
  • Богатый экосистема
  • Много готовых open-source решений

Минусы

  • Только Windows
B

Кросс-платформа (Windows + Linux + macOS)

Единый код, работает везде. Но глубокая интеграция с ОС сложнее, защита в реальном времени ограничена.

Плюсы

  • Широкая аудитория
  • Один кодовый базис

Минусы

  • Ограниченная глубина защиты
  • Больше сложности
C

Windows + Linux

Золотая середина. Windows — основная цель вирусов, Linux — популярен у разработчиков и серверов.

Плюсы

  • Покрывает большинство случаев
  • Реально реализуемо

Минусы

  • Всё равно два разных стека для глубокой интеграции
